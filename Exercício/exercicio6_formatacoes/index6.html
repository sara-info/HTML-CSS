Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Netsta frase, temos um termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica).

Nesta frase, temos um termo em ênfase usando a tag EM (semântica).

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag INS (semântica).

Existe também o sublinhado com a tag U (não semântica)

Texto sobrescrito

Para inserir coisas como x20 + 3 usando a tag SUP.

Texto subscrito

Para inserir coisas como H2O usando a tag SUB.

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto grande e texto pequeno

Estamos criando um texto grande usando a tag BIG e um texto pequeno usando a tag SMALL nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando a tag DEL.
